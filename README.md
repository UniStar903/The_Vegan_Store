# The Vegan Store

A static vegan store landing page built with HTML and CSS. This project demonstrates a single-page responsive UI with navigation, hero section, feature cards, customer testimonials, stats, video section, and contact form layout.

## 🌿 Project Overview

This is a lifestyle storefront demo page for a vegan brand. It includes:
- Fixed top navigation with dropdown menu
- Hero banner with background image and tagline
- Four-step feature cards
- Product value cards
- Testimonials section
- Follow-us subscription panel over an image
- Store statistics and sponsor video section
- Contact/address information with a simple form
- Mobile responsive behavior for smaller widths

## 📁 Files

- `index.html` — Landing page structure and content
- `Project.css` — Styling and responsive layout rules
- Image and media assets used by the page:
  - `tree.png`, `scissors.png`, `brush.png`, `bag.png`, `smile.png`, etc.
  - `veganbgimg.jpg`, `strawberry.jpeg`, `One Punch Man 3 Trailer.mkv`

## 🚀 How to run

1. Open `index.html` in your browser (double-click or use VS Code Live Server).
2. Browse the sections and resize the window to test responsiveness.
